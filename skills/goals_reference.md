## Skill: Справочник целей Яндекс Метрики

Используется при любых вопросах про цели, goal_*, конверсии, лиды, события на сайте.

Цели хранятся в двух местах:
- `dm_traffic_performance` — колонки `goal_XXXXXXX` (счётчик сессий, в которых сработала цель)
- `dm_client_journey` — поле `goals_in_visit` (Array(UInt32)), проверка: `has(goals_in_visit, GOAL_ID)`

---

## Готовые агрегаты для SQL

```sql
-- Все лиды (основная цель):
SUM(goal_314553735) AS all_leads

-- Только звонки-лиды (качественные):
SUM(goal_201619843 + goal_201619846 + goal_201619840 + goal_63191746) AS call_leads

-- Только формы:
SUM(goal_313904512 + goal_338849075 + goal_314247265) AS form_leads

-- CRM воронка:
SUM(goal_332069613) AS crm_created,
SUM(goal_332069614) AS crm_paid

-- CR в лид (очищенный от спама):
round(SUM(goal_314553735) / nullIf(SUM(visits) - SUM(goal_402733217), 0) * 100, 2) AS lead_cr_pct

-- Вовлечённость (без лида):
SUM(goal_330230879) AS favorites,
SUM(goal_321286953) AS quiz_starts
```
